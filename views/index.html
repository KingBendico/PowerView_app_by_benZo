Important Information

To ensure the proper functionality of the PowerView app, you must enter the IP address of your local network device.

How to Enter the IP Address:

  1. Click on the Settings Button: Locate the settings button in the top-right corner of the application window. This button is represented by a cogwheel icon (⚙️).
  2. Open the Settings Window: Clicking the settings button will open the settings window.
  3. Enter the IP Address: In the settings window, you will find a field labeled "IP Address." Please enter the IP address of your local network device in this field. The IP address should be in the format 192.168.1.1.
  4. Save the IP Address: After entering the IP address, click the "Save" button to store this information. The application will use this IP address to connect to your local network device.
  5. Close the Settings Window: Once the IP address is saved, you can close the settings window.

Where to Find the PowerView Primary Gateway IP:

  1. Check Your Router's GUI: Access your router's graphical user interface (GUI) to view all connected devices. Look for a device named "PowerView-G3." If there is more than one PowerView gateway, the primary gateway is typically the one with the lower IP address (e.g., 192.168.0.101 is likely the primary gateway compared to 192.168.0.102).
  2. Use the Browser: Type or copy this URL into your browser: http://powerview-g3.local/gateway/swagger?enable=true. This will enable/start the Swagger UI for the PowerView gateway. Note that it uses port 3002, while the execution server uses port 80.
